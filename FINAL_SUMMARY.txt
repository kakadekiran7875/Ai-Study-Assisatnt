⚡ QUICK SETUP (3 STEPS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  Get OpenAI API Key
    → https://platform.openai.com/api-keys
    → Click "Create new secret key"
    → Copy key (starts with sk-...)
    → Add to .env: OPENAI_API_KEY=sk-...

2️⃣  Get Supabase Anon Key
    → https://supabase.com/dashboard/project/qzhhekukwoqltqxrwxlx/settings/api
    → Copy "anon public" key (starts with eyJ...)
    → Add to .env: SUPABASE_ANON_KEY=eyJ...

3️⃣  Run Database Schema
    → Go to Supabase SQL Editor
    → Copy/paste from src/config/schema.sql
    → Click "Run"

4️⃣  Restart Server
    → Press Ctrl+C
    → Run: npm run dev
